# Workshop 1 – ISW711 – Primera API con Express

Proyecto correspondiente a la **Semana 1**, donde se desarrolló una API básica utilizando **Node.js** y **Express**, integrando conexión a **MongoDB Atlas**, con el fin de comprender la estructura inicial de una API y el manejo de rutas.

---
## Tecnologías utilizadas

- Node.js
- Express
- MongoDB Atlas
- Postman

---

## Instalación
1. Abrir la terminal en la carpeta del proyecto.
2. Instalar dependencias

`npm install`

3. Crear un archivo `.env` en la raíz del proyecto y agregar la cadena de conexión a MongoDB:

`env`
DATABASE_URL=string_de_conexion_mongodb

---

## Ejecución

- Para iniciar el servidor, ejecutar:
`npm start`
- El servidor se ejecuta en: 
http://localhost:3000/
