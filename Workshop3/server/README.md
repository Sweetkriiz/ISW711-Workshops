Sistema de Gestión de Profesores y Cursos

Proyecto que implementa una REST API con Node.js, Express y MongoDB para administrar profesores y cursos, junto con un frontend básico en HTML y JavaScript que consume dicha API.

---
## Tecnologías

* Node.js + Express
* MongoDB + Mongoose
* HTML + JavaScript (Fetch API)

---

## Cómo ejecutar

1. Instalar dependencias en `server`:

```
npm install
```

2. Crear archivo `.env` con la URL de MongoDB

3. Iniciar el backend:

```
npm start
```

4. Abrir `client/index.html` en el navegador
